**`metadata.json`:**

A JSON file containing metadata for each scene.

```json
{
    "scene_name": {
        "description": "A brief description of the scene.",
        "duration": "Total duration of the scene in seconds.",
        "frames": [
            {
                "frame_id": 0,
                "body_pos": [

                ],
            },
            ... // other frames
        ]
    }
}
```
